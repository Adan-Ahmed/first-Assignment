# first API endpoint

A simple Flask backend project with two JSON API endpoints. This project demonstrates the basic request–response cycle using Flask and how to return JSON data from a web server.

## Features

* Two JSON API endpoints (`/` and `/about`)
* Built with Python and Flask
* Returns JSON responses using `jsonify()`
* Beginner-friendly backend project

## Run the Project

```bash
pip install -r requirements.txt
python app.py
```

Then open:

* `http://127.0.0.1:5000/`
* `http://127.0.0.1:5000/about`
